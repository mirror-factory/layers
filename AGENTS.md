## Verification Loop (OODA)

After making changes, run this loop until all checks pass:

1. `pnpm typecheck` -- Fix any type errors
2. `pnpm test` -- Fix any failing tests
3. `npx eslint .` -- Fix any lint errors
4. `ai-dev-kit tool validate` -- Fix any registry issues
5. If any check fails, fix and repeat from step 1
6. Maximum 5 iterations -- if still failing after 5, report remaining issues

Do NOT mark a task as complete until all checks pass.
