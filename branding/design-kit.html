The brand is not the marketing site. The brand is the recording session card on the live page — soft paper, a heavy navy timer, the breathing wave, a calendar tile that feels like a real torn-page date, the mint "connected" pill. Every other branded asset is downstream of this.

02 — Typography

System sans. Light weights for body. Heavy for hero numbers.

The product uses the system stack at native rendering — no web-font flash, no upscale serif. Emphasis comes from color and weight, not font-family changes.

Hero — Display weight
Your context, layered.
System sans, weight 600, letter-spacing −0.02em. Heavy 800 reserved for big numerals (the recording timer).
UI — System sans, light to medium
Tap once. Layered for life.
-apple-system → SF Pro Text → Inter → ui-sans-serif. Body: weight 400 / 500. Labels: 600. Headings: 600. Hero numerals: 800.
Mono — tool calls, transcripts, code reveals
layers.search({ query: "pricing decisions", since: "30d" })
JetBrains Mono → SF Mono → Menlo → ui-monospace. Same family the explainer video uses (see remotion/scenes/Mcp.tsx).

08 — Voice

Calm, specific, portable, inevitable.

From branding/BRAND_NARRATIVE.md §7.

Do say

  • "Your meeting memory in every AI tool."
  • "Tap once. Layered for life."
  • "Your context, layered."
  • "Claude knew about Thursday. We didn't have to tell it."
  • Specific verbs: capture, layer, recall, hand off, route, surface.
  • Always name destinations: Claude, ChatGPT, Gemini, Cursor.

Don't avoid

  • "Layers is the future of knowledge work." (too grand)
  • "Magically gives your AI context." (cliché & dodges the verb)
  • "Unlock the power of…" (cliché stack)
  • "Revolutionary / 10x / supercharge / mind-blowing."
  • Pretending Layers is the destination instead of the substrate.
  • AI sparkles on every surface. One flourish per layout.

10 — Using this kit

One file. Share it. Drop tokens straight in.

For designers

Send this file. It opens by double-click. Every token name maps 1:1 to DESIGN.md, app/globals.css, and remotion/lib/tokens.ts. Copy SVGs straight from the logo board.

For agents

Read branding/BRAND_NARRATIVE.md first, then DESIGN.md. Never paste raw hex into product code. Use the semantic OKLCH tokens. For canvas/shader code, the rgba wave colors are listed in §4 above.
